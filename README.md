How to start the app locally:

1.clone this app to your desktop
```
cd desktop
git clone https://github.com/zheyuyansam/cs103a-cpa02
```
2. cd to this app
```
cd cs103a-cpa02
```
3. install npm and start the app
```
npm install
npm start
```
4. Open http://localhost:5000/
